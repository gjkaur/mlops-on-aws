# Project template companions (Lab 10)

AWS **SageMaker Studio Projects** draw **Service Catalog** products—CloudFormation executes the authoritative template AWS ships for your landing zone. Folder **`cloudformation/companion-demonstration.yaml`** is a **narrow** stack for whiteboard contrasts (CodeCommit + S3 + Model Package Group) when you deliberately want resources **outside** a Studio-managed project.

Folder **`seed-code/`** hosts a **`buildspec.sample.yml`** plus trivial **pytest** to illustrate how Enterprises gate merges—drop into a cloned template repo only after reading that repo’s norms.
